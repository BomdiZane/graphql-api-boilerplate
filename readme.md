# BOB test API

Test API for BOB

## Run App

```javaScript
// Install dependencies
npm i

// Start server
npm start
```

## Run Tests

API tests will be done with graphiql, the following commands are just placeholders which may be used later.

```javaScript
npm test

// Test coverage
npm run test:coverage
```
